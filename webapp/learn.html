١ مقدمة عن لغة ضاد

لغة ضاد هي لغة برمجة عربية كاملة تعتمد على الكلمات العربية الأصيلة كأوامر برمجية. كل شيء فيها — الكلمات المفتاحية، أسماء المتغيرات، والمخرجات — يكتب بالعربية.

✅

أسماء المتغيرات يجب أن تكون بالحروف العربية فقط (نطاق يونيكود ٠٦٢٠–٠٦٥F)

ℹ️

يمكن استخدام الأرقام الغربية (0–9) أو الأرقام العربية (٠–٩) في القيم العددية

⚠️

كل جملة يجب أن تنتهي بفاصلة منقوطة: ؛ أو ;

٣ المتغيرات

لتعريف متغير، استخدم الكلمة المفتاحية متغير يليها الاسم ثم القيمة:

لغة ضاد 
// تعريف متغيرات من أنواع مختلفة
متغير رقم = 42؛
متغير نص = "مرحبا بالعالم"؛
متغير حالة = صح؛
متغير ناتج = 3.14؛

لتغيير قيمة متغير موجود مسبقاً، اكتب اسمه مباشرة بدون كلمة "متغير":

لغة ضاد 
متغير عداد = 0؛
عداد = عداد + 1؛  // تحديث القيمة

٤ الطباعة — اطبع

استخدم اطبع() لعرض أي قيمة على الشاشة:

لغة ضاد 
اطبع("مرحبا!")؛
اطبع(2 + 3)؛
متغير س = 10؛
اطبع(س)؛

المخرجات: مرحبا! ثم 5 ثم 10

٥ الشروط — إذا / وإلا

لغة ضاد 
متغير درجة = 75؛

إذا (درجة >= 90) {
    اطبع("ممتاز")؛
} وإلا {
    إذا (درجة >= 60) {
        اطبع("ناجح")؛
    } وإلا {
        اطبع("راسب")؛
    }
}

عوامل المقارنة المتاحة:

  • > أكبر من  |  < أصغر من
  • >= أكبر أو يساوي  |  <= أصغر أو يساوي
  • == يساوي  |  != لا يساوي

٦ الحلقات — بينما

لغة ضاد 
// طباعة الأعداد من 1 إلى 5
متغير عداد = 1؛

بينما (عداد <= 5) {
    اطبع(عداد)؛
    عداد = عداد + 1؛
}
⚠️

تأكد من تحديث متغير الشرط داخل الحلقة لتجنب الحلقة اللانهائية

٧ العمليات الحسابية

لغة ضاد 
متغير أ = 10؛
متغير ب = 3؛

اطبع(أ + ب)؛  // 13  — جمع
اطبع(أ - ب)؛  // 7   — طرح
اطبع(أ * ب)؛  // 30  — ضرب
اطبع(أ / ب)؛  // 3.333... — قسمة

٨ أمثلة كاملة

}

مثال ٢: حساب مجموع الأعداد

لغة ضاد 
متغير مجموع = 0؛

    عداد = عداد + 1؛
}
اطبع(مجموع)؛  // 5050
